关于 GitHub 爬虫工具

这是一个基于 MCP GitHub 的贡献者查询工具，能够：

  • 获取 GitHub 项目的详细信息
  • 分析项目贡献者排名
  • 展示贡献者的详细资料
  • 提供美观的可视化界面

技术栈：FastAPI + MCP GitHub + 现代化前端

使用帮助

如何使用：

  1. 在搜索框输入 GitHub 项目名称（格式：用户名/项目名）
  2. 选择要显示的贡献者数量
  3. 点击"开始分析"按钮
  4. 等待数据分析完成，查看结果

示例项目：

  • microsoft/vscode
  • facebook/react
  • vuejs/vue
  • angular/angular
